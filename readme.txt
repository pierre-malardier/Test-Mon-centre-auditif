Voici le travail que j’ai réalisé.
Pour installer le projet vous devez télécharger le zip de git ou alors cloner le répertoire.
Ensuite effectuer la commande composer install dans votre terminal pour que toutes les dépendances et packages que j’ai utilisés s'installent.
Vous devez lancer votre serveur local et effectuer la commande symfony serv.
Un url apparaîtra dans votre console et vous pourrez aller voir ce que j’ai produit.

J’ai réalisé cette page en récupérant votre logo ainsi que la photo présente en fond sur votre site pour que cela se rapproche le plus d’une vraie page.

Comme le sujet l’indiqué j’ai créé un formulaire à l’aide de symfony comprenant les champs demandés.
J’ai établi des contraintes sur le nom et le prénom qui ne doivent pas avoir le même nombre de caractères.
J’ai également placé une contrainte sur l’email pour vérifier que ce dernier est valide ainsi que pour le numéro de téléphone.
La date doit également être après la date actuelle.
Lorsque l’on valide le formulaire, les données de ce dernier sont stockées en base de données. 
Vous êtes redirigé sur une page confirmant que la demande à bien été prise en compte et que vous êtes attendu au lieu de dépôt sélectionnée à la date sélectionnée avec la tranche horaire qui correspond.
J’ai joins la base de donnée que j’ai exporté  dans le répertoire git, le fichier se nomme SAV.sql
Je n’ai pas réalisé la dernière contrainte dans le temps imparti mais compte bien le faire dans les jours qui suivent et allez encore plus loin.
